# Open Bank Transactions

```sh
 $ git clone https://github.com/balu-sag/openbank_transactions.git
 $ cd openbank_transactions
 ```

```sh
$ cd mvn spring-boot:run
```

# Test
```sh
$ mvn test
```

$ curl http://localhost:8080/accounts/savings-kids-john/transactions
